# simple-blog-app-using-reactjs

> Simple Blog App using ReactJS

## How to run

Please follow the below instructions to run this project in your machine:

1. Clone this repository
   ```sh
   https://github.com/c0der-himel/simple-blog-app-using-reactjs.git
   ```
2. Go into cloned directory
   ```sh
   cd simple-blog-app-using-reactjs
   ```
3. Install dev dependencies
   ```sh
   npm install
   ```
4. Run the json server in your terminal:
   ```sh
   npx json-server --watch data/db.json --port 5000
   ```
5. Run the app in your terminal:
   ```sh
   npm start
   ```
6. Your app should be available in http://localhost:3000
